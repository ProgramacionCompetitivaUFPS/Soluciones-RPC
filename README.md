### Soluciones RPC
Este proyecto se creo para funcionar como una biblioteca con las soluciones de los problemas de las maratones RPC, esta divido por años y cada una sigue la numeracion de todas las maratones de ese año, cualquiera puede aportar sus propias soluciones a dichos problemas realizando un fork, para luego pedir un pull request para agregarlas.

### Reglas
* TODOS los archivos agregados deben tener una extensión que indique el lenguaje utilizado (.java, .cpp).
* Cada Archivo tiene que tener la letra del problema al que corresponde.
* Se admiten diferentes soluciones al mismo ejercicio. Si las dos soluciones son en el mismo lenguaje, no es necesario cambiar el nombre, nombrar los archivos como "(Letra)V2", o V3, o la versión que corresponda.
* Se admite dentro del código todos los comentarios inline ( // ) y multiline ( /*  */ ) que considere pertinentes.
* Cada solución, independiente del lenguaje debe contar con la siguiente meta-descripción justo antes de iniciar el código: [Metadescripción](https://github.com/ProgramacionCompetitivaUFPS/soluciones/blob/master/firma.java)

### Inspiracion
Basado en el [proyecto](https://github.com/ProgramacionCompetitivaUFPS/soluciones) creado por los antiguos lideres en 2015, realizamos esta nueva version enfocada en las maratones RPC apartir de 2022.
